storjtelehash module¶

class storjtelehash.StorjTelehash(broadcast_handler, port)[source]¶

Bases: storj.messaging.Messaging

Concrete Messaging layer for Storj Platform in Telehash. Everything in telehash-C is not thread safe. So run function after stop a thread, and run a thread again in all functions.
